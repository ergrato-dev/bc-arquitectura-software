# 📚 Ebooks Gratuitos — Seguridad en Arquitectura de Software

## 🔐 Seguridad y OWASP

### 1. **OWASP Top Ten (2021) — Guía Oficial**

- **Autor**: OWASP Foundation
- **Descripción**: Documento oficial de las 10 vulnerabilidades más críticas en aplicaciones web. Incluye descripción, ejemplos y mitigación para cada categoría.
- **Link**: [owasp.org/Top10](https://owasp.org/Top10/)
- **Nivel**: Intermedio
- **Idioma**: Inglés (con traducciones parciales)

### 2. **OWASP Testing Guide v4.2**

- **Autor**: OWASP Foundation
- **Descripción**: Guía completa de pruebas de seguridad web. Cubre autenticación, autorización, inyección, manejo de sesiones y más.
- **Link**: [owasp.org/Testing Guide](https://owasp.org/www-project-web-security-testing-guide/)
- **Nivel**: Avanzado
- **Idioma**: Inglés

### 3. **OWASP Cheat Sheet Series**

- **Autor**: OWASP Foundation
- **Descripción**: Colección de hojas de referencia rápida sobre autenticación, JWT, CSRF, SQL Injection, etc. Ideal para consulta diaria.
- **Link**: [cheatsheetseries.owasp.org](https://cheatsheetseries.owasp.org/)
- **Nivel**: Principiante-Avanzado
- **Idioma**: Inglés

## 🏗️ Arquitectura Segura

### 4. **The Web Application Hacker's Handbook** (Fragmentos libres)

- **Autores**: Stuttard & Pinto
- **Descripción**: Referencia clásica para entender cómo piensan los atacantes. Fundamental para diseñar defensas efectivas.
- **Disponibilidad**: Fragmentos en O'Reilly Free Trial
- **Nivel**: Intermedio-Avanzado
- **Idioma**: Inglés

### 5. **Node.js Security Best Practices (Node.js Team)**

- **Autor**: Node.js Foundation
- **Descripción**: Guía oficial de mejores prácticas de seguridad para aplicaciones Node.js: secretos, validación, autenticación, rate limiting.
- **Link**: [nodejs.org/en/docs/guides/security](https://nodejs.org/en/docs/guides/security)
- **Nivel**: Intermedio
- **Idioma**: Inglés

### 6. **Microservices Security in Action** (Capítulos gratuitos)

- **Autor**: Prabath Siriwardena
- **Descripción**: Seguridad en arquitecturas de microservicios: OAuth 2.0, JWT, mTLS, API Gateways. Los primeros capítulos están disponibles gratis.
- **Link**: [manning.com/books/microservices-security-in-action](https://www.manning.com/books/microservices-security-in-action)
- **Nivel**: Avanzado
- **Idioma**: Inglés

---

## 🔑 JWT y OAuth

### 7. **JWT.io — Introduction to JWTs**

- **Autor**: Auth0 / Okta
- **Descripción**: Introducción completa a JSON Web Tokens: estructura, algoritmos de firma, cuándo usarlos y cuándo no. Incluye debugger interactivo.
- **Link**: [jwt.io/introduction](https://jwt.io/introduction)
- **Nivel**: Principiante
- **Idioma**: Inglés

### 8. **OAuth 2.0 Simplified**

- **Autor**: Aaron Parecki
- **Descripción**: Guía accesible al protocolo OAuth 2.0. Cubre todos los flujos con diagramas claros. Versión web gratuita completa.
- **Link**: [oauth.net/2/](https://www.oauth.com/)
- **Nivel**: Intermedio
- **Idioma**: Inglés

### 9. **OAuth 2.0 Security Best Current Practice (RFC 9700)**

- **Autor**: IETF
- **Descripción**: RFC con las mejores prácticas actuales para implementar OAuth 2.0 de forma segura, incluyendo PKCE.
- **Link**: [tools.ietf.org/html/rfc9700](https://datatracker.ietf.org/doc/html/rfc9700)
- **Nivel**: Avanzado
- **Idioma**: Inglés
